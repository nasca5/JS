select 태그 다루기셔셔
select 태그를 이용하여 data를 받고싶을 때 주의해야 할 점이 있다.
만약 js로 if문을 이용해 select 태그의 임의의 밸류값이 내가 찾는 값과 같으면 어떠어떠한 문장을 실행하겠다. 라는 식으로 코드를 짜게 되면
이론상 잘 될 것 같지만, 실제로는 잘 되지 않는다. 
그 이유는 바로 평소 js 코드는 동기 처리로 진행되기 때문인데, js 파일은 비동기 처리를 따로 하지 않는 이상 로드된 직후 바로 모든 스크립트를 실행하게 된다.
따라서 내가 어떠한 옵션을 선택할 시간도 없이 그 위의 if문이 실행되어버리기 때문에 내가 원하는 대로 실행이 되지 않는다.
그래서 select 태그같은 내가 무언가를 고르고 나서 실행돼야 할 요소들이 있다면 비동기 처리를 해줘야 하는데, select 태그같은 요소는 이벤트 리스너를 통해 change, input 같은 event를 조건으로 걸어두면
원하는 대로 잘 실행되는 것을 볼 수 있다. 
별다른 비동기 처리 함수가 없다면 js 파일은 항상 바로바로 실행된다는 점을 잊지 말자.
